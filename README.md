# README #

This project implement a pattern miner for colored strings.

**Version:** v 1.0.0

# REQUIRED PACKAGES #
* sdsl-lite
* sqlite3
* libsqlite3-dev

# COMPILE

Export the sdsl-lite library path, e.g.,

`export SDSL_LIBRARY_PATH=../../sdsl-lite `

Move in the build directory

`cd build`

Compile the executables

`make all`

# UTILITIES

* sqlitebrowser
